CCFBLOG(Golang development)
==== 
基于beego框架Golang语言开发的博客
演示地址：http://blog.lcsg.pw/
-------
#Install

##Installation
`go get github.com/who246/myblog`
##config
modify /conf/app.conf
##build
`go build`
##run 
./myblog
